Как это работает
  1. Введите значения коэффициентов для обоих игроков (геймы 5-10)
  2. Приложение вычисляет сумму только десятичных частей каждого коэффициента
  3. Пример: Для 3.45 в сумму добавляется только 0.45
  4. Игрок с меньшей суммой объявляется победителем
